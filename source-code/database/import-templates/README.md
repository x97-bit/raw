# قوالب الاستيراد الرسمية

هذه القوالب مهيأة على أساس **بنية نظام الراوي الحالية**، وليس على أساس ملفات قديمة أو Access.

الملفات الأساسية:

- `alrawi-system-import-templates.xlsx`
- `template-manifest.json`
- ملفات CSV المنفصلة لكل شيت

القواعد المهمة:

- لا تغيّر أسماء الشيتات.
- لا تغيّر أسماء الأعمدة.
- استخدم التاريخ بصيغة `YYYY-MM-DD`.
- استخدم العملة بالقيم: `USD` أو `IQD` أو `BOTH`.
- استخدم نوع الحركة بالقيم: `فاتورة` أو `سند`.
- في شيت `الحركات الموحدة` استخدم القسم بالقيم:
  - `port-1`
  - `port-2`
  - `port-3`
  - `transport-1`

أهم الشيتات:

- `الحركات الموحدة`: القالب الرسمي العام لكل الحركات
- `السعودية - حركات`
- `المنذرية - حركات`
- `القائم - حركات`
- `النقل الداخلي`
- `ديون - باسم الجميلي`
- `ديون - حيدر (أنابيب)`
- `ديون - لؤي`
- `ديون - نومان (صيرفة)`
- `خاص - حيدر شركة الأنوار`
- `خاص - ياسر عادل`
- `جدول المحافظات`

متى نستخدم أي قالب؟

- إذا كانت بياناتك متنوعة ومختلطة: استخدم `الحركات الموحدة`
- إذا كانت بياناتك منظمة حسب القسم: استخدم الشيت المخصص لذلك القسم
- للديون والحسابات الخاصة: استخدم الشيتات المخصصة لها، لأنها تتبع جداول مختلفة عن `transactions`

هذه القوالب **فارغة عمدًا** حتى تكون مرجعًا نظيفًا للاستيراد القادم.
